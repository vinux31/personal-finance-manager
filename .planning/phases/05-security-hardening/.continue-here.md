## Critical Anti-Patterns (do NOT repeat these)

- **ANTI-PATTERN**: Spawning multiple Agent calls with `isolation=worktree` in a single message → `.git/config.lock` contention. **Mitigation**: dispatch one Agent per message, with `run_in_background: true`, letting tool-call latency space out the worktree creates.
- **ANTI-PATTERN**: Modifying STATE.md/ROADMAP.md inside a worktree executor → last-merge-wins overwrites. **Mitigation**: Worktree mode in execute-plan.md auto-skips shared-file updates; orchestrator owns those writes after wave completes.
