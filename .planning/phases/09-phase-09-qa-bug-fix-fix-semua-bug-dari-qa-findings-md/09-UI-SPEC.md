## Interaction States

Hanya state yang berubah di Phase 9:

### Bug #4 — Auth Toast Error
- **Trigger:** `onAuthStateChange` event menerima `SIGNED_OUT` atau error refresh token (AuthApiError "Refresh Token Not Found")
- **Visual:** Toast `sonner` dengan type `error` — ikon `OctagonXIcon`, warna `--destructive`
- **Copy:** "Sesi berakhir, silakan login kembali"
- **Duration:** Default sonner duration (4000ms)
- **Follow-up:** `supabase.auth.signOut()` dipanggil → redirect ke halaman login

### Bug #6 — Helper Text GoalDialog
- **Trigger:** Selalu tampil (tidak conditional) di bawah input "Dana Kas Terkumpul"
- **Visual:** `<p className="text-xs text-muted-foreground">` — konsisten dengan pattern di baris 106 yang sudah ada
- **Placement:** Setelah `<Input id="g-current" .../>`, dalam `<div className="grid gap-2">`

### Bug #7 — DialogDescription Accessibility
- **Trigger:** Selalu tampil (render statik di dalam `<DialogHeader>`)
- **Visual:** `<DialogDescription>` mengikuti style shadcn default — `text-sm text-muted-foreground`
- **Placement:** Setelah `<DialogTitle>`, sebelum penutup `</DialogHeader>`
