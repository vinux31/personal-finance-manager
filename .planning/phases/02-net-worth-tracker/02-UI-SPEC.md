## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge padding, inline label gaps |
| sm | 8px | Internal card padding rows, gap between badge and text |
| md | 16px | Default element spacing, dialog field gap, card padding |
| lg | 24px | Section gap between "Aset & Rekening" and "Liabilitas" sections |
| xl | 32px | Tab-level vertical rhythm between summary card and sections |
| 2xl | 48px | Chart container top margin |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Touch targets for edit/delete icon buttons: minimum 44px hit area (p-2.5 wrapper, `h-4 w-4` icon) — consistent with GoalsTab action buttons.
- Summary Net Worth card inner padding: `p-4` (16px) — consistent with MetricCard and GoalsTab summary bar.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (regular) | 1.5 |
| Label / sublabel | 10px (text-[10px]) | 600 (semibold) | 1.5 |
| Heading (section title) | 16px (text-base) | 600 (semibold) | 1.2 |
| Display (Net Worth total) | 20px (text-xl) | 600 (semibold) | 1.1 |

Source: Existing MetricCard uses `text-[10px] font-semibold uppercase tracking-wider` for labels and `text-xl` for values — replicated with semibold weight for consistency.

Notes:
- Investasi read-only row label: `text-sm italic text-muted-foreground` with a small "otomatis" badge.
- Chart x-axis tick labels: `text-sm` (14px), weight 400, `text-muted-foreground` color.
- Dialog form field labels: `text-sm font-semibold` (14px, 600).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--background)` — oklch(1 0 0) light / oklch(0.145 0 0) dark | Page background, tab background |
| Secondary (30%) | `var(--card)` — oklch(1 0 0) light / oklch(0.205 0 0) dark | Account/liability cards, dialog background, section panels |
| Accent (10%) | `#6366f1` (brand indigo) | Net Worth summary card gradient, section add-buttons, chart stroke/fill |
| Destructive | `var(--destructive)` — oklch(0.577 0.245 27.325) | Delete account button, delete liability button, ConfirmDialog |

Accent reserved for:
1. Net Worth summary gradient card background: `linear-gradient(135deg, #6366f1, #818cf8)` (D-07, matches "Net Bulan Ini" card).
2. Dashboard MetricCard 5 (Net Worth): same gradient — `gradient` prop passed as `true`.
3. Area chart stroke color and gradient fill: stroke `#6366f1`, fill gradient from `#6366f1` (opacity 0.3) to transparent.
4. "Tambah Akun" and "Tambah Liabilitas" primary buttons — uses shadcn `Button` default (brand primary via `--primary`).
5. Account/liability card left-border accent: `borderLeft: '4px solid var(--brand)'` — consistent with GoalCard pattern.

Secondary semantic colors (non-accent):
- Investasi read-only row: no color accent — plain card with italic label and "otomatis" badge (variant="secondary").
- Trend badge up: `bg-emerald-100 text-emerald-700`.
- Trend badge down: `bg-red-100 text-red-700`.

Source: D-07, `src/index.css` brand tokens, `src/tabs/DashboardTab.tsx` MetricCard implementation.

## Layout Contracts

### KekayaanTab — Mobile (< 640px)

```
[ Summary Card: Net Worth / Aset / Liabilitas ]    ← gradient indigo card, full width

[ Section Header: "Aset & Rekening" ][ + Tambah Akun ]

[ Account Card 1 ]   ← full width, rounded-xl border bg-card, borderLeft brand
[ Account Card 2 ]
[ Investasi Row ]    ← read-only, italic label, "otomatis" badge, no edit/delete buttons

[ Section Header: "Liabilitas" ][ + Tambah Liabilitas ]

[ Liability Card 1 ]
[ Liability Card 2 ]

[ Section Header: "Tren Net Worth" ]

[ AreaChart — 6 bulan ]   ← full width, 200px height
```

### KekayaanTab — Desktop (>= 768px)

Accounts and liabilities each remain single-column stacked cards (Claude's discretion per CONTEXT.md: 1 column at all breakpoints — matching GoalsTab's `grid-cols-1 gap-4 md:grid-cols-2` only for goals, but since Net Worth items are rows not cards with progress bars, single column is cleaner for scannability).

Summary card: full width.
Chart: full width.

### Summary Card (top of KekayaanTab)

Three-number layout inside gradient card:
```
Net Worth                              ← label: text-[10px] semibold uppercase tracking-wider text-indigo-200
Rp 450.000.000                         ← value: text-2xl font-semibold

Aset  Rp 500.000.000   Liabilitas  Rp 50.000.000   ← row of two, text-sm text-indigo-200
```

Source: CONTEXT.md `<specifics>` — "tampilkan 3 angka dalam satu gradient card di atas, seperti summary bar di GoalsTab".

### Account / Liability Card Row

```
[ Icon ]  Name (text-sm font-semibold)     Type badge (variant="outline", text-xs)
          Saldo: Rp X (text-xs muted)                          [ Pencil ] [ Trash2 ]
```

Left border: `borderLeft: '4px solid var(--brand)'` — consistent with GoalCard.
Card padding: `p-4`.
Icon: Lucide `Landmark` for tabungan/giro, `Wallet` for cash/dompet, `Building2` for properti, `Car` for kendaraan/cicilan kendaraan, `CreditCard` for kartu kredit/paylater/KTA, `Home` for KPR.

Accessibility labels on icon-only action buttons:
- Edit account: `aria-label="Edit akun {nama}"`
- Delete account: `aria-label="Hapus akun {nama}"`
- Edit liability: `aria-label="Edit liabilitas {nama}"`
- Delete liability: `aria-label="Hapus liabilitas {nama}"`

### Investasi Read-Only Row

```
[ TrendingUp ] "Nilai Investasi"  <Badge variant="secondary">otomatis</Badge>
               Rp X (text-xs muted)                          (no action buttons)
```

Row uses `italic` on the name label. No left border accent (visually distinct from editable rows).

### Dashboard MetricCard 5 (Net Worth)

Uses existing `MetricCard` component with `gradient={true}` and `trend={netWorthTrend}` props.

```
NET WORTH                ← label text-[10px] semibold uppercase tracking-wider text-indigo-200
Rp 450.000.000           ← text-xl font-semibold text-white
  ↑ 3%                   ← trend badge (emerald if positive, red if negative) — hidden if null
```

Dashboard grid: `grid-cols-2 sm:grid-cols-3 md:grid-cols-5` — update to accommodate 5th card. The existing `sm:grid-cols-4` wraps to accommodate the new card.

Source: D-06, D-07, MetricCard implementation in DashboardTab.tsx line 175+.

### Net Worth Trend AreaChart

```
[ Select: 3 bulan / 6 bulan / 12 bulan ]   ← right-aligned, h-8, above chart

[ AreaChart — ResponsiveContainer width="100%" height={220} ]
  X-axis: month label (Jan, Feb, Mar…) — text-sm muted-foreground
  Y-axis: shortRupiah values — text-sm muted-foreground
  Tooltip: formatRupiah full value + bulan label
  Area stroke: #6366f1 strokeWidth={2}
  Area fill: url(#netWorthGradient) — linearGradient from #6366f1@0.3 to #6366f1@0
```

Source: D-08, D-09, ReportsTab.tsx AreaChart pattern.

## Dialog Contracts

### NetWorthAccountDialog

Title: "Tambah Akun" (new) / "Edit Akun" (editing)

| Field | Component | Validation |
|-------|-----------|------------|
| Nama Akun | Input, placeholder "cth: BCA Tabungan Utama" | Required, min 1 char |
| Tipe | Select | Required. Options: tabungan, giro, cash, deposito, dompet digital, properti, kendaraan |
| Saldo / Nilai | Input type="text" inputMode="numeric" | Required, must be > 0, formatted as number |

CTA buttons:
- Cancel: `Button variant="outline"` — label "Batal"
- Submit: `Button` (default/primary) — label "Simpan"

### NetWorthLiabilityDialog

Title: "Tambah Liabilitas" (new) / "Edit Liabilitas" (editing)

| Field | Component | Validation |
|-------|-----------|------------|
| Nama | Input, placeholder "cth: KPR BTN Rumah Cipete" | Required, min 1 char |
| Tipe | Select | Required. Options: KPR, cicilan kendaraan, kartu kredit, paylater, KTA |
| Jumlah Outstanding | Input type="text" inputMode="numeric" | Required, must be > 0 |

CTA buttons:
- Cancel: `Button variant="outline"` — label "Batal"
- Submit: `Button` (default/primary) — label "Simpan"

Both dialogs follow GoalDialog pattern: controlled state in parent, `open` + `onOpenChange` + `editing` props, `useMutation` + `toast.success`/`toast.error` inside dialog.

## Interaction Contracts

| Interaction | Trigger | Behavior |
|-------------|---------|----------|
| Open add-account dialog | Click "+ Tambah Akun" | `setEditing(null); setDialogOpen(true)` — same pattern as GoalsTab |
| Open edit-account dialog | Click Pencil icon on account row | `setEditing(account); setDialogOpen(true)` |
| Open add-liability dialog | Click "+ Tambah Liabilitas" | `setEditingLiability(null); setLiabilityDialogOpen(true)` |
| Open edit-liability dialog | Click Pencil icon on liability row | `setEditingLiability(liability); setLiabilityDialogOpen(true)` |
| Delete account | Click Trash2 → ConfirmDialog → confirm | `deleteAccount.mutate(id)` → `toast.success("Akun dihapus")` |
| Delete liability | Click Trash2 → ConfirmDialog → confirm | `deleteLiability.mutate(id)` → `toast.success("Liabilitas dihapus")` |
| Auto-snapshot on tab open | `useEffect` with `[]` deps on KekayaanTab mount | Call `insertSnapshotIfNeeded(uid, currentMonthKey)` — idempotent |
| Hover chart point | Recharts `<Tooltip>` | Show month label + `formatRupiah(value)` in tooltip |
| Investasi row — no hover action | Read-only row | No cursor-pointer, no hover border change, Pencil/Trash2 not rendered |
