# QA Audit pfm-web Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Temukan dan dokumentasikan semua bug di 10 halaman pfm-web ke dalam `QA-FINDINGS.md`, siap dijadikan basis GSD Phase 09 Bug Fix.

**Architecture:** Dua-pass per halaman — (1) static code audit baca source file cari bug logic/SQL/error-handling; (2) live browser test via Playwright untuk konfirmasi bug dan temukan issue yang tidak terlihat dari kode. Semua temuan dikumpulkan ke satu file `QA-FINDINGS.md`.

**Tech Stack:** React + TypeScript, Supabase (PostgreSQL triggers + RLS), Vite dev server, Playwright MCP untuk browser testing.
